TAGS- Estruturais
<html> ... </html> A tag HTML, abreviação de HyperText Markup Language, é usada para estruturar o conteúdo de uma página da web.
<head> ... </head> Essa tag delimita o cabeçalho do documento. Não possui nenhum valor visível, porém é capaz de transmitir ao navegador diversas informações muito úteis e essenciais.
< body > ... </body> Essa tag representa o corpo do documento,é nessa tag que todos os elementos visíveis do seu site devem ser inseridos.
<title> ... </title> Define um título para o documento.
<div> ... </div> Define uma seção em um documento.


TAGS - Listas
<ul> ... </ul> Define uma lista não ordenada.
<ol> ... </ol> Define uma lista ordenada.
<li> ... </li> Define um item de lista.
<reversed> ... </reversed> Faz alterações nas especificações padrão das listas, alterando valores dos atributos e, desse modo,personalizando-as.
<start>...</start> Faz alterações nas especificações padrão das listas, alterando valores dos atributos e, desse modo,personalizando-as.


TAGS - Semânticas
<main> ... </main> A tag MAIN especifica o conteúdo principal e, consequentemente, de maior relevância dentro da página.
<header> ... </header> Essas tags definem um cabeçalho. Portanto, todo conteúdo que estiver dentro dela faz parte de um cabeçalho, podendo ser utilizado dentro de outras sessões.
<footer> ... </footer> Define um rodapé para um documento ou seção.
<article> ... </article> Define um artigo.
<aside>...</aside> Define o conteúdo além do conteúdo da página.
<section>...</section> Define uma seção em um documento.
<figure>...</figure> Especifica conteúdo independente.
<figcaption>...</figcaption> Define uma legenda para um elemento <figure>
<nav>...</nav> Define links de navegação.
<wbr> Define uma possível quebra de linha.


TAGS - Texto
<h1 ... h6> ... </h1 ... h6> Define títulos HTML.
<hgroup> ... </hgroup> Define um cabeçalho e conteúdo relacionado.
<p> ... </p> Define um parágrafo.
<b> ... <b/> Define um texto em negrito.
<i>...</i> e <s>...</s> Define uma parte do texto em uma voz ou humor alternativo.
<br/> Define uma única quebra de linha.


TAGS - Links e âncoras
<a href=””> Nome da âncora </a> Define um hiperlink.
<a href=mailto:> É usado para criar um link que, quando clicado, abre o cliente de e-mail do usuário com um endereço de e-mail específico já preenchido.
<a href=”tel://#####-####”>Número de telefone</a> É usado para criar um link clicável que permite aos usuários ligar para um número de telefone diretamente de uma página da web.
<a name=”name”> ... </a> É usado para especificar um nome para um elemento HTML. Isso permite que o elemento seja referenciado por um hyperlink ou por um script.


TAGS - Imagens
<img/> Define uma imagem.
src="URL" É um atributo usado para especificar o endereço URL de um recurso externo que você deseja incluir na sua página web.
alt="texto" É usado em elementos de imagem (<img>) para fornecer um texto alternativo para a imagem, caso ela não possa ser exibida.
height="altura da imagem" Define a altura de uma imagem.
width="largura da imagem" Define a largura de uma imagem.
border=” ...” É usado para definir a largura da borda de um elemento, como uma tabela ou imagem.
<área/> Define uma área clicável dentro de um mapa de imagem (um <map>). É usada para criar links diferentes para diferentes partes de uma imagem.
shape="..." É usado para definir a forma de uma área clicável em um mapa de imagem.
coords="..." É usado para definir as coordenadas de uma área específica dentro de um mapa de imagem.
<map> ... </map> Define um mapa de imagem.
